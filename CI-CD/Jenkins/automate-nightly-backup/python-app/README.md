### PyNightlyBackup

#### The project automates the daily backup of Python applications, including code, configurations, and data. Using Jenkins and S3, it ensures reliable and regular storage of the latest versions of artifacts, allowing you to easily restore the application in case of unexpected errors or data loss. Ideal for CI/CD processes and nightly backups.
